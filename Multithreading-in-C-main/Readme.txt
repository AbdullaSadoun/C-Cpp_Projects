Purpose
Utilize multithreading in c to create a program to read a list of integers from a file, split them and sort them parallely then merge them. 

Objective
Use Multithreading and get familiar with pthread in C.


to run the program:
1- navigate to the direcotry in the terminal that has the main.c and IntegerList.txt files
2- type in: 
    gcc main.c -lpthread -o main
3- type in: 
    ./main IntegerList.txt
4- Check the SortedIntegerList.txt file to view the output of the program
