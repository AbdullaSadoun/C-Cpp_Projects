Word Separator

Purpose of the code:
Demonstrate knowledge of using a linux virtual machine to test and run your code by making a simple word separating program that prints the output to an output file.

How to use: 
1- Enter in the terminal: “gcc -o main main.c” then hit enter
2- Type in “./main.c” then hit enter, you will be prompted to enter a sentence.
3- Enter your sentence then press enter or return.
4- Check the output file “separatedwords.txt” to see the output.
