To run the program:

1 - Navigate to the directory where the main.c file is located using cd

2 - type in "gcc -o BoundedBuffer main.c -lpthread -lrt"

3 - type in "./BoundedBuffer <arg1> <arg2> <arg3>"
    where "<argn>" is replace by your actual arguments 
    sample input: "./BoundedBuffer 10 2 3"
